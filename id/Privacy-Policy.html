1. Informasi yang Kami Kumpulkan

Di CaliperTools, kami berkomitmen untuk melindungi privasi Anda. Kami mengumpulkan jenis-jenis informasi berikut:

  • Data Penggunaan: Kami dapat mengumpulkan statistik penggunaan anonim untuk meningkatkan layanan kami.
  • Informasi Peramban: Kami dapat mengumpulkan jenis peramban, versi, dan informasi sistem operasi untuk tujuan kompatibilitas.
  • File Log: Kami dapat mengumpulkan log kesalahan dan log sistem untuk debugging dan meningkatkan layanan kami.

2. Cara Kami Menggunakan Informasi Anda

Kami menggunakan informasi yang dikumpulkan untuk tujuan-tujuan berikut:

  • Perbaikan Layanan: Untuk menganalisis pola penggunaan dan meningkatkan alat dan layanan kami.
  • Dukungan Teknis: Untuk mendiagnosis dan menyelesaikan masalah teknis.
  • Keamanan: Untuk mendeteksi dan mencegah aktivitas berbahaya.

3. Pemrosesan Data

Semua pemrosesan data terjadi secara lokal di peramban Anda. Kami tidak menyimpan atau mentransmisikan file Anda ke server kami kecuali secara eksplisit diperlukan oleh layanan kami.

  • Pemrosesan Lokal: Sebagian besar alat kami memproses file Anda langsung di peramban Anda tanpa mengunggah ke server mana pun.
  • Tidak Ada Penyimpanan Data: Kami tidak menyimpan file atau data pribadi Anda di server kami.
  • Penyimpanan Sementara: Setiap data sementara dihapus segera setelah pemrosesan.

4. Berbagi Data

Kami tidak menjual, menyewakan, atau berbagi informasi pribadi Anda dengan pihak ketiga untuk tujuan pemasaran.

  • Tidak Ada Berbagi Pihak Ketiga: Kami tidak berbagi data Anda dengan pihak ketiga untuk pemasaran.
  • Penyedia Layanan: Kami dapat menggunakan layanan pihak ketiga terpercaya untuk fungsi tertentu (misalnya, analitik).
  • Kewajiban Hukum: Kami dapat mengungkapkan informasi jika diwajibkan oleh hukum.

5. Keamanan Data

Kami menerapkan langkah-langkah keamanan yang sesuai untuk melindungi informasi Anda:

  • Enkripsi: Kami menggunakan enkripsi standar industri untuk transmisi data.
  • Protokol Aman: Kami menggunakan HTTPS dan protokol aman lainnya.
  • Pembaruan Rutin: Kami secara rutin memperbarui praktik keamanan kami.

6. Hak Anda

Anda memiliki hak-hak berikut terkait informasi pribadi Anda:

  • Koreksi: Anda dapat meminta koreksi atas informasi yang tidak akurat.
  • Penghapusan: Anda dapat meminta penghapusan informasi pribadi Anda.
  • Pilihan Keluar: Anda dapat memilih keluar dari praktik pengumpulan data tertentu.

7. Cookie

Kami menggunakan cookie dan teknologi serupa untuk meningkatkan pengalaman Anda:
