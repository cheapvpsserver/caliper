Panduan Langkah demi Langkah

Langkah 1: Unggah File Video Anda

Anda dapat mengunggah video dengan dua cara:

  • Klik pada area unggahan dan pilih file video dari komputer Anda
  • Seret dan lepas file video langsung ke area unggahan

Penampil mendukung format video MP4 dan AV1 untuk analisis metadata.

Langkah 2: Lihat Metadata Video

Setelah video Anda diunggah, penampil akan secara otomatis menganalisisnya dan menampilkan metadata terperinci, termasuk:

  • Info File: Nama, ukuran, jenis MIME, dan tanggal modifikasi terakhir
  • Wadah: Merek utama, merek kompatibel, skala waktu, durasi, dan status fragmentasi
  • Trek Video: Kodek, resolusi, bitrate, dan rotasi
  • Warna: Warna primer, karakteristik transfer, koefisien matriks, dan kedalaman bit

Anda dapat menyalin metadata ke papan klip atau mengunduhnya sebagai file JSON untuk analisis lebih lanjut.

Langkah 3: Putar Video dan Ambil Frame

Gunakan pemutar video untuk melihat pratinjau video Anda. Anda dapat:

  • Memutar, menjeda, dan mencari melalui video
  • Mengambil frame di titik mana pun dalam video
  • Mengunduh frame yang diambil sebagai gambar PNG

Fitur penangkapan frame memungkinkan Anda mengekstrak gambar berkualitas tinggi dari file video Anda.

Kiat untuk Hasil Terbaik
  • Gunakan file video berkualitas tinggi untuk ekstraksi metadata yang lebih akurat
  • Pastikan file video Anda tidak rusak atau dienkripsi
  • Untuk file video besar, beri sedikit waktu agar analisis selesai
  • Gunakan fitur penangkapan frame untuk mengekstrak momen penting dari video Anda
