Tutorial Langkah demi Langkah

Langkah 1: Unggah File Video Anda

Anda dapat mengunggah video dengan dua cara:

  • Klik pada area unggahan dan pilih file video dari komputer Anda
  • Tarik dan jatuhkan file video langsung ke area unggahan

Pemeriksa mendukung semua format video utama termasuk MP4, WebM, dan MKV.

Langkah 2: Analisis Informasi Kontainer Video

Setelah video Anda diunggah, pemeriksa akan secara otomatis menganalisisnya dan menampilkan informasi terperinci, termasuk:

  • Info File: Nama, ukuran, format kontainer, tipe MIME, dan durasi
  • Aliran Video: Resolusi, bitrate perkiraan, dan frame rate
  • Aliran Audio: Saluran, tingkat sampel, dan bitrate perkiraan
  • Kompatibilitas: Dukungan browser dan rekomendasi format

Anda juga dapat melihat pratinjau video secara langsung di browser.

Langkah 3: Ekspor dan Bagikan Hasil

Gunakan opsi ekspor untuk menyimpan atau berbagi informasi kontainer video Anda:

  • Salin metadata ke clipboard untuk berbagi cepat
  • Unduh sebagai JSON untuk tujuan pengembangan
  • Unduh sebagai teks untuk kemudahan membaca

Pemeriksa menyediakan wawasan kompatibilitas untuk membantu Anda memilih format terbaik untuk kebutuhan Anda.

Kiat untuk Hasil Terbaik
  • Gunakan file video berkualitas tinggi untuk analisis yang lebih akurat
  • Pastikan file video Anda tidak rusak
  • Untuk file video besar, beri waktu untuk menyelesaikan analisis
  • Periksa informasi kompatibilitas untuk dukungan antar-browser
