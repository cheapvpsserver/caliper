Tutorial Langkah demi Langkah

                    

                        

                            
Langkah 1: Masukkan Teks Anda

                            
Anda dapat memasukkan teks dengan dua cara:

                            
    
                                
  • Langsung tempel teks Anda ke area teks sebelah kiri
    • 
                                
  • Klik "Pilih File" untuk mengunggah satu atau lebih file TXT
    • 
                            

                            
Setiap baris teks akan diperlakukan sebagai satu item untuk digabungkan.

                        

                        
                        

                            
Langkah 2: Konfigurasikan Opsi Penggabungan

                            
Sesuaikan bagaimana teks Anda digabungkan:

                            
    
                                
  • Mode Penggabungan: Pilih pemisah (spasi, koma, baris baru, atau kustom)
    • 
                                
  • Opsi Pembersihan: Abaikan baris kosong dan pangkas spasi
    • 
                                
  • Duplikat: Hapus baris duplikat dengan sensitivitas huruf besar/kecil opsional
    • 
                                
  • Urutkan: Urutkan baris secara alfabetis (A-Z atau Z-A)
    • 
                                
  • Awanama/Akhiran: Tambahkan awanama dan akhiran kustom ke setiap baris
    • 
                            

                        

                        
                        

                            
Langkah 3: Dapatkan Hasil Anda

                            
Teks gabungan Anda akan muncul di area output sebelah kanan secara instan. Anda dapat:

                            
    
                                
  • Salin teks yang digabungkan dengan satu klik menggunakan tombol "Salin"
    • 
                                
  • Bersihkan semua teks untuk memulai ulang dengan tombol "Bersihkan"
    • 
                            

                            
Area output secara otomatis menambahkan bilah gulir untuk konten yang panjang.

                        

                        
                        

                            
Tips untuk Hasil Terbaik

                            
    
                                
  • Untuk file besar, pertimbangkan untuk membaginya menjadi potongan-potongan yang lebih kecil
    • 
                                
  • Gunakan opsi "Abaikan baris kosong" untuk membersihkan input Anda
    • 
                                
  • Coba berbagai pemisah untuk menemukan format terbaik untuk kebutuhan Anda
    • 
                                
  • Gunakan opsi awanama/akhiran untuk menambahkan tanda kutip atau karakter lain ke setiap baris
    •
